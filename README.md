# Ulus Haber WEB(HTTP) API

Bu proje, Ulus Haber biriminin [Discord Duyuru Kanalları](https://ulus.org.tr/discord-haber-kanalları) projesini aşmak için geliştirdiği bir alt yapı projesidir.

Şimdilik sadece Ulus Haber tarafından yayınlanan haberler için tasarlanıyor, ama [gelecek planları](./ROADMAP.md#gelecek-planları) arasında [çoklu yayıncı mimarisi](./ROADMAP.md#çoklu-yayıncı-mimarisi) bulunuyor.
